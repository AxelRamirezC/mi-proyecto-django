# Proyecto Django

## Descripción
Proyecto web desarrollado con Django como parte de la tarea final de Topicos. Incluye página de inicio y 'Acerca de' con diseño Tailwind CSS.

## Características
- Framework: Django
- Estilos: Tailwind CSS
- Arquitectura por componentes
- Sistema de plantillas reutilizables
- Interfaz adaptable

## Instalación

### Prerrequisitos
- Python 3.8+
- pip

### Pasos de instalación

1. **Clonar el repositorio**
```bash
git clone https://github.com/AxelRamirezC/mi-proyecto-django.git
cd mi-proyecto-django
```

2. **Crear y activar entorno virtual**
```bash
# Windows
python -m venv venv
venv\Scripts\activate
```

3. **Instalar dependencias**
```bash
pip install -r requirements.txt
```

4. **Ejecutar migraciones**
```bash
python manage.py migrate
```

5. **Ejecutar servidor**
```bash
python manage.py runserver
```

6. **Abrir en navegador**
```
http://127.0.0.1:8000
```
